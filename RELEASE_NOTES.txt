BalanceIQ v9.8.3 — Demo Dashboard & PWA Cache Fix

Fixed
- Installed versions could continue serving the older v9.8.1 app.js because the service worker cache name had not been updated in v9.8.2.
- Demo Mode now verifies that sample transactions are present after saving.
- Dashboard filters are cleared and all dashboard/report views are recalculated after loading demo data.

Improved
- Demo transactions have stable IDs, explicit demo markers, and numeric amounts.
- Demo removal reliably targets only demo records.
- Storage failures now show a clear error instead of silently leaving the dashboard empty.
