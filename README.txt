BalanceIQ v9.6.1 — Smart Data Management

A private, browser-based finance app with fast assisted receipt entry.

Version 9.6.1 adds:
- Create and rename Assets, Categories and Sub-categories without leaving transaction entry or receipt review
- Asset assignment during receipt capture
- In-context Asset creation during merchant review
- Safe reference migration when names are changed
- Polished in-app management dialogs instead of browser prompts
