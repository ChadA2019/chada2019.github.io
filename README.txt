BalanceIQ v9.2 — Streamlined Assisted Receipt Entry

Deploy every file in this folder together.

Receipt workflow:
1. Photograph or select the receipt.
2. Select Quick Scan.
3. BalanceIQ performs one fast local OCR pass and fills likely values.
4. Check the highlighted fields, correct anything inaccurate, then save.

The development-only OCR diagnostics panel has been removed to keep receipt entry simpler and cleaner. Quick Scan remains intentionally assistive rather than exhaustive. All OCR runs locally in the browser.
